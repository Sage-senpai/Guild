## 1. Active Protocol Grants (High Priority)

### Filecoin Foundation / Protocol Labs
- **Program**: Filecoin Foundation Developer Grants
- **Size**: $5,000 – $250,000 per grant
- **Focus**: Storage applications, DataDAOs, tooling, consumer apps
- **Fit for Guild**: HIGH — agent knowledge bases as decentralized data, DataDAO pattern for knowledge curation
- **Application**: Open rolling applications at grants.filecoin.io
- **Typical timeline**: 4-6 weeks decision; milestone-based disbursement
- **2026 status**: Active; Filecoin Foundation continues to prioritize storage applications

### Arbitrum DAO LTIPP / STIP
- **Program**: Long-Term Incentive Pilot Program
- **Size**: $10,000 – $3M in ARB tokens
- **Focus**: DeFi, consumer apps, infrastructure on Arbitrum
- **Fit for Guild**: HIGH — deploy credit registry + agent NFTs on Arbitrum; LTIPP rewards user activity
- **Application**: Forum proposal → snapshot vote → DAO approval
- **Typical timeline**: 4-8 weeks for approval; ongoing emission period
- **2026 status**: LTIPP rounds continue; new funding tranches expected Q1-Q2 2026

### Optimism RPGF (Retroactive Public Goods Funding)
- **Program**: RetroPGF rounds (currently Round 6 in progress)
- **Size**: $10M – $100M per round distributed to badgeholders
- **Focus**: Public goods — infrastructure, education, tooling, open-source
- **Fit for Guild**: MEDIUM-HIGH — if Guild is positioned as open-source public good infrastructure for AI agents
- **Strategy**: Build in public; get users; apply to RPGF round 6+
- **Unique**: No upfront application — prove impact first, get funded retroactively
- **2026 status**: RPGF Round 5 results in early 2026; Round 6 expected mid-2026

### Web3 Foundation Grants (Polkadot)
- **Program**: Open Grants Program (formerly Grants Program)
- **Size**: $10,000 – $100,000 per milestone
- **Focus**: Polkadot/Kusama ecosystem tools, SDKs, infrastructure
- **Fit for Guild**: MEDIUM — if building a Substrate parachain for the marketplace
- **Application**: GitHub PR to w3f/grants program repository
- **Typical timeline**: 3-4 weeks decision; strict milestone tracking
- **2026 status**: Active; ~$50M+ total disbursed historically; continues

### Polygon Village / Foundation
- **Program**: Polygon Village Grants + Ecosystem Fund
- **Size**: $5,000 – $500,000
- **Focus**: DeFi, NFTs, gaming, consumer apps on Polygon zkEVM / CDK
- **Fit for Guild**: MEDIUM-HIGH — Polygon has best Africa RPC coverage; CDK = dedicated chain
- **Application**: village.polygon.technology
- **2026 status**: Active; accelerator cohorts ongoing

### Solana Foundation
- **Program**: Solana Foundation Developer Grants + Superteam Grants
- **Size**: $5,000 – $100,000
- **Focus**: Consumer apps, DeFi, gaming, infrastructure
- **Fit for Guild**: MEDIUM — Superteam Africa chapter provides local support and grants
- **African angle**: Superteam Nigeria, Kenya, Ghana chapters = local community + grant access
- **Application**: airtable.com/shrRbw6bJYCi0xGIb (Foundation); superteam.fun (Superteam)
- **2026 status**: Active; Superteam Africa growing rapidly

### Near Foundation / DevHub
- **Program**: Near Horizon accelerator + DevHub grants
- **Size**: $10,000 – $250,000
- **Focus**: Consumer apps, B.O.S. components, DeFi
- **Fit for Guild**: MEDIUM — BOS component deployment for agent marketplace UI
- **Application**: DevHub on-chain forum (dev.near.org)
- **2026 status**: Active

### Celestia Foundation
- **Program**: Celestia Ecosystem Builder Program
- **Size**: $10,000 – $500,000
- **Focus**: Rollups using Celestia DA, infrastructure, tooling
- **Fit for Guild**: MEDIUM — if building a sovereign rollup using Celestia DA

## 2. Hackathon Bounty Landscape (2026)

### ETHGlobal
- **Events**: Bangkok, London, NYC, San Francisco (quarterly)
- **Bounty range**: $1,000 – $100,000 per sponsor prize
- **Sponsors most relevant**: Arbitrum, Base, Optimism, Filecoin, Arweave, EigenLayer
- **Strategy**: Enter AI + Web3 track; target multiple sponsor prizes simultaneously
- **Guild's strongest angle**: Storage proofs + AI inference = targets both infra and AI tracks

### Solana Hackathons (Colosseum)
- **Events**: Radar (online), Breakpoint (annual), regional
- **Bounty range**: $100,000 – $1,000,000 total prize pool per event
- **Best fit**: Consumer track + Africa track (via Superteam bounties)
- **2026**: Colosseum's Radar hackathon expected Q2 2026

### DoraHacks (Global)
- **Platform**: Continuous bounties + hackathons
- **Relevant programs**: Filecoin/IPFS track, Polkadot track, AI track
- **Tip**: DoraHacks has significant African community; submit via hackerlink.io

### Gitcoin Grants (Rounds)
- **Type**: Quadratic funding rounds
- **Focus**: Open-source, public goods
- **Strategy**: If Guild open-sources its core, apply to Climate + Open Source + Web3 Community round
- **2026**: Gitcoin Grants 23+ expected Q1, Q3 2026

## 3. Africa-Specific Funding Opportunities

### Superteam Africa
- **Communities**: Nigeria, Kenya, Ghana, South Africa chapters
- **Grants**: $1,000 – $25,000 for Africa-focused Solana projects
- **Support**: Local mentorship, co-working, demo days
- **Requirements**: Must have Africa-specific use case or team member

### Africa Web3 Foundation (AWF)
- **Focus**: Blockchain education + startup acceleration in Africa
- **Grants**: $5,000 – $50,000 for African founders
- **Countries**: Pan-African focus; Nigeria, Kenya strongest

### Binance Labs x Africa
- **Program**: BNB Chain MVB Accelerator (Most Valuable Builder)
- **Size**: Up to $500,000 in BNB tokens + investment
- **Focus**: DeFi, GameFi, SocialFi on BNB Chain
- **Note**: Requires BNB Chain integration

### Google for Startups / Accelerator: Africa
- **Type**: Non-dilutive + $200K Google Cloud credits
- **Focus**: AI-powered startups with African focus
- **Relevance**: Guild's AI agent + Africa angle fits perfectly
- **Application**: Twice annually; highly competitive

### European Development Bank (EIB) / UNDP Digital Finance
- **Type**: Innovation grants + technical assistance
- **Focus**: Financial inclusion technology in developing markets
- **Relevance**: Credit-based economy for underbanked users = strong fit
- **Scale**: $100K – $2M; longer timeline (3-6 months)
